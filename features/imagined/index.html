Features in the Imagined test scope scope

The Imagined test scope scope has the following features defined. These are available to assign to rulesets to build a view of answerability of the competencies. Available rulesets are listed on the front page of this site

imagined.feature.current

Currently available feature

imagined.feature.future

Potentially available feature

imagined.feature.unlikely

Feature that will never be available

imagined.feature.enabler

Feature that enables another feature

imagined.feature.transitive

Feature enabled by another feature
